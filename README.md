# Machine Learning Math Foundations

This repository is dedicated to learning and practicing the **mathematical foundations required for Machine Learning**.  
The goal is to build a strong understanding of the core mathematical concepts that power modern ML algorithms.

## Purpose

Machine Learning relies heavily on mathematics.  
This repository serves as a **personal study space** to learn, implement, and experiment with the key mathematical topics used in ML.

The focus is on understanding both:
- the **theory**
- the **practical implementation**

## Topics Covered

The repository is organized around the following fundamental areas:

### 1. Linear Algebra
Linear algebra is the backbone of machine learning and deep learning.

Topics include:
- Vectors
- Matrices
- Matrix multiplication
- Linear transformations
- Eigenvalues and eigenvectors
- Singular Value Decomposition (SVD)

### 2. Calculus
Calculus is used to understand how models learn and optimize.

Topics include:
- Derivatives
- Partial derivatives
- Gradients
- Chain rule
- Optimization with gradient descent

### 3. Probability
Probability is essential for modeling uncertainty in machine learning.

Topics include:
- Random variables
- Probability distributions
- Conditional probability
- Bayes' theorem
- Expectation and variance

### 4. Statistics
Statistics helps analyze data and evaluate machine learning models.

Topics include:
- Descriptive statistics
- Sampling
- Hypothesis testing
- Confidence intervals
- Statistical inference

### 5. Optimization
Optimization techniques are used to train machine learning models.

Topics include:
- Cost / loss functions
- Gradient descent
- Stochastic gradient descent
- Convex optimization basics
- Regularization techniques

## Repository Structure

The repository will be organized by topic:
	linear-algebra/
	calculus/
	probability/
	statistics/
	optimization/

Each section may contain:
- notes
- code implementations
- exercises
- experiments

## Goal

The objective of this repository is to build a **solid mathematical intuition for machine learning** and create a reference for future ML and AI projects.
